# whyfi
Euron 7기 와이파이(WhyFi) : 금융 용어 알리미 팀의 레포지토리입니다


### straemlit 실행
1. 
    ```(bash)
    $ streamlit run streamlit.py
    ```
2. 
    http://localhost:8501 접속

### chrome extension 실행
1. 
    ```(bash)
    $ python3 app.py
    ```
2. 
    chrome://extensions/에서 개발자 모드로 변경 후 `chrome-extension` 폴더 로드
